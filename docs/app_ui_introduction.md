## 2) Catalog Biofrench

Use **Catalog Biofrench** to browse the Biofrench product catalog.

### Small icons (top bar)

On the catalog screen you may see a few small icons in the top bar:

- **Star (⭐) — Show/Hide the “Other” tab**
  - **What it does:** Toggles whether the **Other** source/tab is available.
  - **How to use it:** Tap the **Star** icon once to show the **Other** tab, tap again to hide it.

- **Heart (❤️) — Thank You**
  - **What it does:** Opens the **Thank You** screen.
  - **How to use it:** Tap the **Heart** icon at any time.

### How to use the catalog

- **Switch source**: Use the buttons **Biofrench** / **Affiliate** (and **Other** if enabled) to filter the catalog by source.
- **Search**: Use the search box to quickly find medicines by name.
- **View an item**: Tap a medicine tile to open the image in full screen.
  - In full screen, **swipe left/right** to move between items.
  - Tap the **X (Close)** icon to exit full screen.
